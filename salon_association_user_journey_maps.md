# **Salon Association Platform — User Journey Maps**

---

## **1. Owner Journey Map**
### **Persona:** Salon Owner (Member of the Association)
### **Goals:** Manage salon, employees, finances, inventory, customers, and use agent/loan services.

#### **1.1 Key Stages**
| Stage | Actions | Touchpoints | Emotions | Opportunities |
|-------|---------|-------------|----------|----------------|
| **Discovery** | Learns about platform via association meeting | Website • Association leaders | Curious | Provide clear value proposition |
| **Onboarding** | Registers salon, uploads documents | Mobile app • Web app | Motivated | Guided onboarding + tooltips |
| **Daily Operations** | Track customers, services, products, staff attendance | Dashboard • POS • Inventory module | Empowered | Automate repetitive tasks |
| **Financial Mgmt** | Records expenses, revenue, reviews reports | Accounting module | Confident | Recommend cost-saving insights |
| **Micro‑Lending** | Applies for loan, tracks repayment | Loans module | Relieved | Instant scoring + loan simulations |
| **Agent Services** | Acts as Airtel AgentLite, performs transactions | Airtel sub‑app | Busy but satisfied | Cash flow prediction, fraud alerts |
| **Growth** | Tracks performance KPIs | Reports module | Proud | Personalized business coaching |
